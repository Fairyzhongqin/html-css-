安装less: 1.先安装node.js http://nodejs.cn/download 2.检查是否安装成功，使用cmd命令 window+r 输入node -v即可 3.基于nodejs在线安装less ，使用cmd命令 npm install -g less即可 4.检查是否安装成功,使用cmd命令less -v查看版本 less是一门css预处理语言，扩展了css的动态特性 使用可以新建一个后缀名为less的文件,在这个less文件里面写less语句 less变量 less编译 less嵌套 less运算(可以加减乘除) less文件不能直接被html页面引用，所以必须编译成css文件 在vscode插件里下在easyless插件,就可以把less文件转化成css ,然后再html页面中使用
